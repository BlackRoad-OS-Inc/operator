## What BlackRoad IS (in 60 seconds)

BlackRoad OS is a complete technology stack running on 5 Raspberry Pis and 2 cloud servers that replaces AWS, GitHub, Slack, OpenAI, Cloudflare, and 12 other services — for $38/month.

It includes:
- **52 TOPS of AI inference** (2x Hailo-8 accelerators + Ollama on 4 nodes)
- **273 self-hosted Git repos** on Gitea
- **Self-hosted DNS, VPN, TLS, storage, databases, cache, CI/CD, chat**
- **200+ AI agents** with persistent memory and cryptographic identity
- **20 root domains** and **14 product subdomains**
- **45+ applications** across 10 product tiers
- **90 CLI tools** and **400+ shell scripts**

The only external dependencies are Stripe (card processing) and GoDaddy (domain registrar).
