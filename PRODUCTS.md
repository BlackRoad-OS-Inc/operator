## The 5 Product Pillars

### P1: OS Workspace (app.blackroad.io)
Where individuals and teams live day-to-day. Tasks, notes, AI assist, workspace management. The daily driver.

### P2: Education / RoadWork (edu.blackroad.io)
Adaptive K-12 tutoring. Teach-back mechanism. Free for students. Competency-based, not time-based. The first vertical we're shipping.

### P3: Creator / RoadTube / Studio
Content production OS. Canvas (design), Video (editing), Writing, SoundRoad (music), RoadTube (distribution). 80-90% creator revenue share. Ideas take 5 seconds; production should too.

### P4: Governance / Cece Protocol (gov.blackroad.io)
Policy evaluation, event ledger, delegation, claims. Every critical action passes through: Policy checked -> Delegation verified -> Ledger updated. Governance as a protocol, not a product.

### P5: Mesh & Agents (mesh.blackroad.network)
Physical + digital mesh of agents on Pi fleet. 30K+ agent ceiling. Each agent has cryptographic identity, persistent memory, device awareness, moral reasoning.

## The Window-in-a-Window Paradigm

BlackRoad OS runs entirely in the browser. Every app opens as a draggable, resizable window inside a Desktop OS container. All windows can see each other and share context. Lucidia AI orchestrates everything through conversation.

Example flow:
1. You're building a game in Genesis Road
2. Say: "I need epic battle music for this scene"
3. Cadence window opens, hears what's happening in Genesis Road
4. Creates contextual music based on the scene
5. Auto-imports into your game timeline
6. Ask Lucidia: "Is this level balanced?" -> AI playtests and suggests changes

Zero exports. Zero copy-paste. Zero context switching. 14 tools replaced.
